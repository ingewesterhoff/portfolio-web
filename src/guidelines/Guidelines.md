# Portfolio Guidelines

## Typography

- **H1/H2 Headings**: Big Shoulders Display Bold only
- **H3 Headings/Labels/Buttons**: Outfit Bold only
- **Body Text**: Roboto regular only

## Color Palette

- **Background**: #261E35
- **Accents**: #C4A9FF
- **Text**: #E5DAFF

## Design

- Dark, modern aesthetic with purple glow effects
- Professional and polished
- Responsive design
